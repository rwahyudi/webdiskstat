Help

What This Report Shows

webdiskstat turns a gdu or ncdu scan into a static disk usage report. The left panel lists entries in the current directory. The treemap shows the same directory visually by size.

Directory List

  • Click a row to select it and update the details panel.
  • Double-click a directory to enter it.
  • Use the column headers to sort by name, items, files, size, or modified date.
  • Use the column settings button next to the Name header to show or hide optional columns.
  • Modified time appears when it was included in the scan data.

Treemap

  • Each rectangle is a file or directory in the current directory.
  • Larger rectangles use more disk space.
  • Directory tiles use different shaded colors; files are colored by extension.
  • Hover a tile for path and size. Double-click a directory tile to enter it.

Home View

  • The home view shows the root scan path, a smaller treemap, and a biggest-files list.
  • Choose 10 to 50 biggest files from the list menu. Results are paged 10 at a time.
  • Drag the divider between the treemap and biggest-files list to resize the home panes.
  • Double-click a listed file to jump to the directory containing it.
  • The generated date is shown at the bottom of the report.

Navigation

  • Use the breadcrumb path at the top to jump to a parent directory.
  • Drag the divider between the directory list and right panel to resize the right panel.
  • Use the parent button next to the Name column header to go up one directory.
  • Use the home icon in the breadcrumb path to return to the scan root.
  • Use the theme switch to toggle between dark and light themes.
  • The URL changes as you browse, so directory views can be bookmarked.
